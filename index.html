Recent Projects

Project 1

AI Based Autonomous Voice Enabled Robot with Real Time Object Detection and Collision Avoidance using Arduino

An AI based voice controlled robot which is controlled via the voice commands/inputs given by the user, acts further accordingly, and also provides features like collision avoidance and object detection with the help of UR sensors and ESP32 camera modules installed in the prototype of the robot architecture. This robot is best suited for mostly on-ground applications like in defence services, where it can be used to detect landmines, granades, and other weapons, because of which the physical damage to human life can be reduced, as it streams the live camera screen on the mobile application virtually which makes it easier to monitor through mobile device.

Project 3

U and Me Farming Website in Flutter

A Website developed in Flutter for an startup company u and me farming where the customer can purchase the farming related items as per their need from the website in broad category like fertilizers, pesticides and seeds and we also update them accordingly as per the new item arrives to us. From this website the customer can buy products as well as sell the product online to other customer with the help of the website which serves in this domain. This website mainly focuces on the village peoples who live far away from the KSK (Krushi Seva Kendra's) which are situated in the cities, from this website it reduces the stress of the farmer to sell or buy the farming product.

Live Weather application developed in flutter inside which the user just has to specify the name of the city of which they want to know the temperature and weather, after entering name of the city the temperature of that particular city gets displayed on the screen with the type of weather on that day in that city.
